Commands

Core commands your staff will actually use during sessions.

/startup

Launch a session with server name, code, pings, and notes.

/release

Open the server publicly with details that are easy to scan.

/reinvites

Handle reinvite waves with counts and extra instructions.

/peacetime

Toggle peacetime status cleanly for everyone in-server.

/priority

Show whether priority is available, active, or on cooldown.

/scene

Post active scene information with location-based detail.

/modcall

Request moderator help when RP or tickets need backup.

/lowmod

Alert the team when staff coverage is getting thin.

/fullmod

Mark coverage as filled so extra pings stop quickly.

/ssd

Announce server shutdown in a consistent format.

/staff

Push staff-only notices without writing ad hoc messages.

/embed

Create polished custom embeds directly from the bot.
